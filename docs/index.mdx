**libsql-durable** is the durable kernel-state layer of a log-structured **Process Virtual Machine (PVM)** whose entire world can live in one libSQL file — and whose CPU can be thrown away at any time.

## One sentence

> Open one world file → the whole machine is there: processes, schedule, memory, durability, ops, fork/explore, and promote.

## Install (Rust crate)

```toml
[dependencies]
libsql-durable = { git = "https://github.com/DSamuelHodge/libSQL-durable", default-features = false, features = ["native-libsql"] }
duroxide = "0.1.29"
tokio = { version = "1", features = ["macros", "rt-multi-thread"] }
```

```sh
# Host binary
cargo run --bin pvm --no-default-features --features native-libsql -- health --world ./world.db
```
